## Eclipse
1. project: File > Import > proj.android
2. create: Classes/xxxxModule.cpp & Classes/xxxxModule.h
3. edit: jni/Android.mk
4. compile : python build_native.py
5. install apk

1. project: File > Open > proj.android-studio
2. create: Classes/xxxxModule.cpp & Classes/xxxxModule.h
3. edit: jni/Android.mk
4. compile : python build_native.py
5. install apk
